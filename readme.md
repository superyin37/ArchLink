# 项目说明

## 激活 venv 环境

Windows PowerShell:
```powershell
.\venv\Scripts\Activate.ps1
```

Windows CMD:
```bat
venv\Scripts\activate.bat
```

macOS / Linux:
```bash
source venv/bin/activate
```

## 使用 uv 激活虚拟环境

创建环境 (默认目录 .venv):
```powershell
uv venv
```

激活环境 (Windows PowerShell):
```powershell
.\.venv\Scripts\Activate.ps1
```

激活环境 (Windows CMD):
```bat
.\.venv\Scripts\activate.bat
```

激活环境 (macOS / Linux):
```bash
source .venv/bin/activate
```
